    • Sobre o Projeto

    Este projeto tem como propósito principal introduzir e explorar os fundamentos essenciais do desenvolvimento web, utilizando tecnologias como HTML, CSS e JavaScript. Através dele, os usuários utilizam uma estrutura de páginas de forma semântica, que foram baseadas em estilos modernos e responsivos com o uso de Flexbox e Grid, assim, vale a pena entender a lógica básica por trás da programação para web. O site foi idealizado com a missão de facilitar a criação e personalização de imagens, independentemente da finalidade — seja para uso pessoal, acadêmico ou profissional. O desenvolvimento foi realizado por Matheus Arantes Almeida, estudante da Cesar School, com o apoio e orientação do professor Matheus Garrido. Este site marca seu primeiro projeto prático utilizando as linguagens HTML, CSS e JavaScript, representando um passo importante no início de sua jornada como desenvolvedor.
